New features

  • None.

Known Problems

  • Opening the RDI driver-specific configuration dialog box can lead to a crash if the third-party RDI DLL (eg. Multi-ICE.dll) uses very old Windows components. As a workaround the .rdi-cfg file, located where the RDI DLL is found, can be manually edited. Alternatively, an older version of IAR Embedded Workbench for ARM (before 5.50) can be used to do the configuration part.
    [EW21929]
