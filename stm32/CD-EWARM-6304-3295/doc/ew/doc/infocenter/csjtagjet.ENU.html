User guide corrections

  • C-SPY® Debugging Guide (UCSARM-3), additions:

      Page 43:
      Installing the JTAGjet USB driver

      Before you can use the JTAGjet interface over the USB port, the JTAGjet USB driver must be installed.
      Use the USB cable to connect the computer to the JTAGjet probe.

      Before Windows 7:
        Because this is the first time the JTAGjet interface and the computer are connected, Windows will open a dialog box and ask you to locate the USB driver. The drivers can be found in the product installation in the arm\drivers\jtagjet directory. Once the initial setup is completed, you will not have to install the driver again.
      Windows 7:
        Start the Windows Device Manager.
        Select Other devices/JTAGjet and right-click
        Choose: Update Driver Software
        Choose: Browse my computer for driver software
        Browse to the arm\drivers\jtagjet directory.
        Click NEXT
        Click INSTALL
