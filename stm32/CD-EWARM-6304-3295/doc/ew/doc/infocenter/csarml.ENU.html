V5.50 2010-04-21

Program corrections

  • In EWARM 5.50.5:
    Starting execution from a breakpoint no longer failes when there is a breakpoint also on the next instruction. This earlier resulted with the error message "Unable to start execution 0x00000004"
    [EW21835]

  • In EWARM 5.50.6:
    Implemented the special reset sequence needed by TI Stellaris Tempest/Firestorm devices.
    [EWARM-973]

V5.11 2007-11-28

Program corrections
