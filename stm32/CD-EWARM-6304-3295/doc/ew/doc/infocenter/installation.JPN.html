システム要件

IAR Embedded Workbenchのこのバージョンをインストールして実行するには、以下が必要です：

  • Microsoft Windows XP(SP3)、Vista(SP2)またはWindows 7を搭載したPentium互換のPC。32ビットおよび64ビット両方のWindowsに対応しています。
  • Internet Explorer 7以上
  • 1 GB以上のRAMおよび2 GB以上のディスク空き容量
  • 製品文書にアクセスするためのAdobe Acrobat Reader

使用するWindowsのバージョンのサポートレベルによっては、サードパーティ製デバッガのドライバが機能しないこともあります。

ライセンス管理システム

QuickStartキーを用いて製品をインストールする場合、 キーは30日間使用できます。ライセンスを有効にするため、この期間にIARシステムズに本製品を登録する必要があります。登録後に永久キーが届き、これを使用して有効なライセンスを持つ製品を使用できるようになります。ライセンスサーバまたはドングルを使用しない場合、本製品はインストールされたコンピュータにロックされます。スタンドアローンのライセンスは、ライセンス転送機能を使用して、別のPCに転送できます。
