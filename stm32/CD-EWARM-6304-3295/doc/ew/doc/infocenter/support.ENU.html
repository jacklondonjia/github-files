SUPPORT

If you have problems using the IAR tools, there are several sources of information that can be useful.

Documentation

A rich source of information are the extensive user manuals (including release notes) as well as the online help system. User guides and release notes can both be reached via the buttons on this page, the online help can be reached from within the IAR Embedded Workbench IDE.

Tech Notes

On www.iar.com/support you can find tech notes providing assistance beyond the material that is included in the product. Some tech notes are answers to Frequently Asked Questions.

Support

If you do not already have an established support contact, the recommended way to request support is to use supp.iar.com/Support/Request, at least for the first contact. If you prefer other ways, there is an up-to-date list of contact information on www.iar.com/support.

Try to isolate the problem. A small code example, project settings, and a description about how to reproduce the problem will significantly help us to provide timely support. Make sure to include your name, product name, product version, and license number.
