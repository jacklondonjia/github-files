System requirements

To install and run this version of IAR Embedded Workbench you need:

  • A Pentium-compatible PC with Microsoft Windows XP(SP3), Vista(SP2), or Windows 7. Both 32-bit and 64-bit variants of Windows are supported.
  • Internet Explorer 7 or higher
  • At least 1 Gbyte of RAM, and 2 Gbytes of free disk space.
  • Adobe Acrobat Reader to access the product documentation

Third-party debugger drivers, might or might not work depending on their level of support for the Windows version used.

License management system

When you install the product with a QuickStart key, it will be usable for 30 days. During this period, the product(s) must be registered with IAR Systems to activate the license. After registration, you will receive a permanent key that opens up the product(s) for which you have a valid license. The product will be locked to the computer on which it has been installed, unless a License Server or dongle is used. A stand-alone license can be transferred to a different PC by using the license transfer functionality.
