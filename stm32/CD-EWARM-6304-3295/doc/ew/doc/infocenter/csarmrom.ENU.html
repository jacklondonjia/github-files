Important information

  • This ROM-monitor driver is designed for the IAR Systems NXP LPC210x Kickstart card, Analog Devices ADuC7000 series evaluation board.

  • The IAR Systems NXP LPC210x Kickstart card supports serial RS232 communication.

  • The Analog Devices ADuC702x evaluation board supports serial RS232 communication.

  • See the Quickstart reference for the IAR C-SPY ROM-monitor that helps you to set up the connection.
